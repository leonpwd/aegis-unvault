# aegis-unvault

Easily decrypt and view the contents of a 2FA [Aegis](https://github.com/beemdevelopment/Aegis) vault in your terminal.

## Features

- Local decryption of Aegis vault files
- Minimalist, scrollable TUI display, search enabled to find that one secret easily
- Export decrypted text

## Usage

1. Decrypt your Aegis vault file (exported JSON format)
2. Browse and save the contents with ease

> ⚠️ This project is not affiliated with [Aegis Authenticator](https://github.com/beemdevelopment/Aegis). For vault format details, see their official repository.
